Guía Definitiva: Cómo Crear y Publicar tu Blog con Hugo en GitHub Pages (Paso a Paso)

#github-pages #gohugo
Oct. 28, 2024

En esta guía te voy a mostrar, paso a paso, cómo crear tu propio blog usando Hugo, una herramienta genial para generar sitios estáticos. Además, vas a aprender a configurarlo para que se despliegue automáticamente en GitHub Pages, de forma simple y sin vueltas. ¡Llevá tu blog a la web en cuestión de minutos!

Tabla de Contenido

  1. Prerrequisitos
  2. Instalación de Hugo
  3. Creación de un nuevo sitio
  4. Configuración de GitHub Pages
  5. Despliegue del blog

Prerrequisitos

Antes de empezar, asegurate de contar con lo siguiente:

  1. Git instalado en tu computadora. Podés descargarlo acá.
  2. Hugo instalado. Podés seguir las instrucciones en la web oficial.
  3. Una cuenta en GitHub para poder crear tu repositorio y usar GitHub Pages.
  4. Editor de texto para trabajar con los archivos del proyecto.
